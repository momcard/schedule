아오님의 기획으로 제작된 스케줄 제작 프로그램입니다.

## 설치

NextJS 14 버전으로 개발하였습니다.

```bash
git clone https://github.com/momcard/schedule.git .

npm i

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
